# Neon Snake Multiplayer (Client/Server)

目标：2~4 人网页联机，房间制（自动进入大厅 → 创建房间/输入房间码加入），服务器权威判定，2 分钟计时，死亡复活。

## 运行（本地）

### 1) 安装 Node.js
需要 **Node.js 18+**（推荐 20+）。

### 2) 安装依赖
```bash
cd snake-mp
npm i
```

### 3) 启动服务器
```bash
npm start
```
默认监听：`http://localhost:8787`

### 4) 打开客户端
浏览器打开：`http://localhost:8787`

同一局多玩家：开多个浏览器窗口/手机访问同一地址，创建房间后把房间码发给朋友。

## 部署
- 任意 VPS/云服务器上跑 `npm start`。
- 建议用 Nginx/Caddy 做反向代理并启用 HTTPS（wss）。

## 规则（当前实现）
- 一局 120 秒。
- 2~4 玩家。
- 撞墙/撞到任意蛇身体：死亡，2 秒后复活（重置长度为 3）。
- 吃普通食物：+10 分，长度 +1。
- 最终按分数排名。

> 后续可加：道具、Boss、关卡地图、排行榜、观战、断线重连。
